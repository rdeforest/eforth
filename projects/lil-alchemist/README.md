# How it "works"

## UI

Prompt is NodeJS repl after polluting global with functions for doing stuff.
Each object type is a function and has a plural form for listing. 

  - player ["name"]
  - deck [number]

There are also 'verbs'

  - add (type)

## Models

### Player

  - props
    - name
  - hasMany
    - Deck deck

### Deck

  - hasMany
    - Card card
  - belongsTo
    - Player player

### CardLevel

  - props
    - Number atk, def

### Card

  - props
    - String name
    - Array<CardLevel> levels
    - BlurgEffect blurg
  - hasMany
    - recipe
    - combo

### Recipe

  - props
    - Number research time
  - hasMany
    - Card ingredient
  - belongsTo
    - Card product

### Library

  - hasMany
    - Player player
    - Card discoveredCard
    - Recipe discoveredRecipe
    - Card cardDefinition
